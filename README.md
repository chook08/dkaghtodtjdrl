길이 설정: 사용자가 생성할 암호의 길이를 지정할 수 있어야 합니다. (예: 8자 ~ 64자)
문자 유형 선택: 암호에 포함될 문자 유형을 선택할 수 있어야 합니다.
대문자 (A-Z)
소문자 (a-z)
숫자 (0-9)
특수 문자 (!@#$%^&*()_+-=[]{}|;:,.<>?)
제외 문자 설정: 특정 문자나 문자 집합을 암호에서 제외할 수 있는 기능입니다. (예: 'l', 'O', '0' 등 혼동하기 쉬운 문자 제외)
암호 강도 측정: 생성된 암호의 강도를 실시간으로 측정하여 사용자에게 피드백을 제공합니다. (예: 약함, 보통, 강함, 매우 강함)
여러 개의 암호 생성: 한 번에 여러 개의 암호를 생성할 수 있는 기능입니다.
암호 복사: 생성된 암호를 클립보드에 쉽게 복사할 수 있는 버튼을 제공합니다.
